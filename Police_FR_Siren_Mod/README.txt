Sirène de Police FR
=========================================

Auteur : Routier87
Version : 1.0

Merci d'avoir téléchargé ce mod.

Ce mod ajoute une sirène de police française afin d'améliorer l'immersion dans Euro Truck Simulator 2.

Installation :
1. Copier le fichier du mod dans le dossier "mod" d'Euro Truck Simulator 2.
2. Activer le mod dans le gestionnaire de mods du jeu.
3. Lancer le jeu et profiter du mod.

Bon jeu et bonne route !
